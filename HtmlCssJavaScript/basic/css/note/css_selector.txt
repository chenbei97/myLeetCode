选择器是根据不同需求把不同的标签选出来，这样选出标签以后再设置这些标签的样式

选择器分为基础选择器和复合选择器
    1）基础选择器就是单个选择器
    基础选择器包括: 标签选择器、类选择器、id选择器和通配符选择器
        ① 标签选择器: 为一类标签设定相同的样式，范围比较大，不能设计差异化样式
        ② 类选择器: 想要差异化选择同一类标签的不同标签，可以使用类选择器，在类名前边加上.
            .green{  将所有拥有red类的html元素设为绿色
                color: green;
            }
            使用类选择器的前提是html的代码里定义时需要指定class属性的值为green，这样就可以找到这个标签

            <div class="firstdiv">我是立即换行1</div>
            .firstdiv{
                color: red;
            }

            class可以指定多个，那么每个class名都可以选出这个标签，如果2个类名有相同的属性后者覆盖前者
            类名是用空格分开的，例如<div class="red xiba"></div>
            一般原则是，如果有相同的属性要提取出来单独作为一个类名
        ③ id选择器
            需要代码设置id属性，同时css使用id选择器是使用#定义
                #id名 {
                    color: red;
                }
            id选择器和class选择器区别是，id选择器只能调用1次，因为代码段不允许有相同的id
                <div id="西巴">西巴</div>
                <div id="西巴">八嘎</div> ×
        ④ 通配符选择器
            使用*来通配
            * {
                margin: 0;
                padding: 0;
            }
            通配符选择器一般不需要调用自动给所有元素应用样式，特殊情况下使用

        总结:
            标签选择器 可以选出所有相同标签,比如div,不能差异化选择 使用较多 div{color:red;}
            类选择器 可以选择1个或多个标签,可以根据需求选择 使用非常多 .nav{color:red;}
            id选择器 1次只能选择1个标签,id属性在每个html文档只能出现1次 一般和js搭配使用 #nav{color:red;}
            通配符选择器 选择所有的标签,选择的太多有部分不需要 特殊情况使用 *{color:red;}
