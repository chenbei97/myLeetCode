新的标签、表单和表单属性，但是存在兼容问题，IE9+才支持

以前的布局都是div来做，class来区分header、nav、content和footer

现在新增了语义标签
<header> 头部标签
<nav> 导航栏标签
<article> 内容标签
<section> 文档某个区域，一般作为article的子元素
<aside> 侧边栏标签
<footer> 尾部标签

多媒体标签
<video> 视频
    autoplay:"autoplay" 自动播放
    controls:"controls" 给用户显示播放控件
    width: pixels 像素 设置播放器宽度
    height: pixels 像素 设置播放器高度
    loop: "loop" 循环播放
    preload: auto预先加载视频|none不加载
    src: url 视频url地址
    poster: imgurl 加载等待的画面图片
    muted: 静音播放
<audio> 音频
    支持mp3、wav、ogg
    autoplay:
    controls:
    loop:
    src:

表单属性：
    required: "required" 表单拥有该属性表示内容不能为空，是必填项
    placeholder: 指示文本 表单的提示信息,存在默认值话不显示
    autofocus: "autofocus" 自动聚焦属性
    autocomplete: "on"|"off" 用户字段开始键入时浏览器基于之前的值来显示,默认on
    multiple: "multiple" 可以多选文件提交
