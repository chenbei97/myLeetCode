HTML的常用标签

1) 标题标签 <h1> - <h6>
    html提供了6个等级的网页标题
    <h1> 我是一级标题</h1>
2) 段落标签
    <p> 我是一个段落标签 </p>
3) 换行标签
    <br /> =>break 强制换行，注意是个单标签
    该标签只是简单地开始新的一行，跟段落不同，段落之间是有间距的
4) 文本格式化标签：粗体、斜体、删除线和下划线
    加粗 <strong></strong> 或 <b></b>
    倾斜 <em></em> 或 <i></i>
    删除线 <del></del> 或者 <s></s>
    下划线 <ins></ins> 或 <u></u>
5) 无语义容器标签
    <div></div> division分割分区,独占1行,大容器
    <span></span> 跨度,一行可以多个<span>，小容器
6) 图像标签和路径
    <img src="url"/> 单标签
    src是img标签的必须属性，指定图像文件的路径和文件名
    <img src="../img.jpg" />

    图像标签的其他属性：
        alt 类型：文本 说明：替换文本，图像不能显示时出现的文字
        title 类型：文本 说明：提示文本，鼠标放到图像上显示的文字
        width 类型：像素 说明：设置图像的宽度
        height 类型：像素 说明：设置图像的高度
        border 类型：像素 说明：设置图像的边框粗细
    所有属性的值都是字符串表示的

    关于路径：最好采用相对路径
    同级的目录(文件夹)下 src = "img.jpg"
    下级的目录 src = "images/img.jpg"
    上级的目录 src = "../images/img.jpg"

    绝对路径可以是网络图片的绝对路径也可以是电脑磁盘的绝对路径

7) 超链接标签
    <a></a> 定义超链接，从一个页面链接到另一个页面
    链接的语法格式
    <a href="跳转目标" target="目标窗口的弹出方式">文本或图像 </a>
    href指定链接目标的url地址，是必须属性
    target是指定打开方式，_blank是在新窗口中打开,_parent是自身变成新窗口，_top和_self和_parent差不多效果
        _self的话 是指在本身这个网页窗口来打开新的网页链接
        _parent主要是针对于框架网页中的跳转 如果你一个页面由两三个框架组成你要想整个页面跳转到其它网页链接 目标就需要用_parent了 如果不用这个 就只会使得那一小块框架区域里的网页跳转 而其它的地方则不变
        _top与_self差不很大 但如果用了<!--#include file="..."-->时，_self点击后的超链接就会在<!--#include file="..."-->这个页面上打开而页面其它部分不会改变
            如果用_top 则会整个窗口一起跳转到新的链接网页

    链接的分类：外部链接、内部链接、空链接和下载链接
    外部链接：http大头的网站
    内部链接：html文件
    空链接：<a href="#">首页</a>
    下载链接：zip文件
    网页元素链接：在网页中的各种网页元素，如文本、图片、表格、音频、视频都可以添加超链接
    锚点链接：这个是上下文链接，快速到达页面某个位置， <a href="#id">第2集</a>
            id是每个标签都具有的属性，可以取一个别名，用于锚点链接定位使用
