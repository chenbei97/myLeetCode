字体图标是在 Web 项目中使用的图标字体

轻量级：一个图标字体比一系列图像要小，一旦字体加载图标就会渲染出来减少服务器请求
灵活性：本质是文字，可以随意的改变颜色、产生阴影、透明效果和旋转
兼容性：几乎支持所有浏览器
注意：字体图标不能替换精灵技术，只是对工作中图标部分技术的提升和优化

总结：结构和样式比较简单的用字体图标，否则使用精灵图

常见的字体图标在网上下载即可，然后引入到html页面即可

http://icomoon.io
http:://www/iconfont.cn

不同浏览器支持的字体格式不同，字体图标之所以兼容，因为包含了主流浏览器支持的字体文件
常见的格式：ttf、woff、svg、eot

使用的时候，需要在css样式中全局声明字体，也就是把这些字体文件通过css引入到我们的界面中

@font-face { /*这里是下载的icomoon内的style.css提供的文件用于外部引入使用*/
    font-family: 'icomoon';
    src: url("../icomoon/fonts/fonts/icomoon.eot?vbzgeg");
    src: url("../icomoon/fonts/fonts/icomoon.eot?vbzgeg#iefix") format('embedded-opentype'),
         url("../icomoon/fonts/fonts/icomoon.ttf?vbzgeg") format("truetype"),
         url("../icomoon/fonts/fonts/icomoon.woff?vbzgeg") format('woff'),
         url("../icomoon/fonts/fonts/icomoon.svg?vbzgeg#icomoon") format('svg');
    font-weight: normal;
    font-style: normal;
    font-display: block;
}

如果下载的图标不够用，想添加新的字体图标到原来的字体文件中
可以把压缩包里边的selection.json重新上传，然后选中自己想要的图标，然后重新下载压缩包替换原来的文件即可
上传就是在官网的icomoon.io/app/#/select中选中import icons，导入selection.json再重新选择即可
