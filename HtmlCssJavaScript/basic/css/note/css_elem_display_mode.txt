什么是元素的显示模式？

    网页的标签很多，不同地方用到不同类型的标签，了解他们的特点可以更好的布局网页
    元素显示模式就是元素以什么方式进行显示，比如div自己占一行，一行又可以放多个span

元素显示模式的分类:

    html的元素一般分为块元素和行内元素两种类型
        块元素：h1-h6、p、div、ul、ol、li等
            可以自己占据一行
            高度、宽度、内外边距都可以控制，宽度默认是容器(父级宽度)的100%
            是个容器，有的块级元素里边还可以继续放置块级元素和行内元素
            注意：文字类的元素内不能使用块级元素
                例如p标签虽然是块级元素，但是不能继续放置块级元素，特别是不能再放div，h1-h6也不行
                div内也不能继续放置span、a等行内元素
        行内元素：常见的有a,strong,em,b,i,del,s,ins,u,span等
            相邻行内元素在一行上，一行可以显示多个
            高、宽直接设置是无效的
            默认宽度就是它本身内容的宽度
            行内元素只能容纳文本或者其它行内元素
            注意：链接里边也不能再放链接
                特殊情况链接内可以放块级元素，但是给a转换一下块级模式最安全
        行内块元素：常见的是img、input、td，同时具备块元素和行内元素的特点
            和相邻行内元素在一行，但是他们之间存在空白间隙，一行可以显示多个
            默认宽度是本身内容宽度
            高度、行高和内外边距可以控制


元素显示模式的转换:

    特殊情况，需要一个模式的元素需要另一种模式的特性，比如想增加a的触发范围，点击所在行也能触发
    需要将a从行内元素转为块元素
    display: block |  inline | inline-block
