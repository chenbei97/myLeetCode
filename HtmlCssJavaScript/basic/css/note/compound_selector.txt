复合选择器是基础选择器合成的

包括：后代选择器、子选择器、并集选择器和伪类选择器

1）后代选择器
后代选择器可以选择父元素里面的子元素，写法是把外层标签写在前边，内层标签在后，中间用空格分开
当标签发生嵌套式，内层标签就成为外层标签的孩子
元素1 元素2 元素3 ...{样式声明} =>  ol li{color: pink;}
.ul2 li a{  ul2是类名也是可以的
    text-decoration: none;
    color: aqua;
}
#ol2{ ol2是id名同样ok
    color: red;
}

2）子选择器
只能选择作为某元素的最近一级子元素，也就是嫡长子，使用>号即可
.nav > a{ /*但是只是希望第一个儿子是紫色,需要使用子元素选择器*/
    color: blueviolet;
}

3）并集选择器
可以选择多组标签，同时为他们定义相同的样式，通常用于集体声明
使用逗号分隔，表示一起定义，另外语法上喜欢竖着写
div,
p,
.pig li{
    color:blue;
}

4）伪类选择器
指定特定位置的元素，使用冒号分割
可以向某些选择器添加特殊的效果，比如链接，或者只选择第1个和第n个元素

常用的链接伪类选择器：选择不同状态的链接
a:link 选择所有未被访问的链接
a:visited 选择所有已被访问的链接
a:hover 选择鼠标指针位于其上的链接
a:active 选择活动链接 鼠标按下未弹起的链接
注意：为了确保生效，必须按照link-visited-hover-active的顺序依次声明

focus伪类选择器：
用于选取获得焦点的表单元素，焦点就是光标，一般来说是<input>类表单元素才能获取
因此这个选择器主要是针对表单元素来说，当鼠标焦点在某个表单控件元素，那么相应的背景颜色就发生改变
input:focus{
    background-color: yellow;
}
